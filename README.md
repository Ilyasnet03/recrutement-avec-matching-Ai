# recrutement-avec-matching-Ai
une plateforme web de gestion du recrutement. Elle permettra aux responsables RH de créer des fiches de poste , ajouter des CV de candidats, et d’utiliser une intelligence artificielle pour trouver automatiquement les profils les plus adaptés. L’objectif est de faciliter et d’accélérer le processus de recrutement de manière efficace et moderne
